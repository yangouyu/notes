# mkdocs 笔记使用教程
1. 安装
```bash
pip install mkdocs  // 安装mkdocs
```

2. 启动
```bash 
mkdocs serve  // 运行
```

3. 打包
```bash 
mkdocs build  // 打包静态环境
```

4. 部署
```bash 
mkdocs gh-deploy  // 部署到github
```

5. 配置文件
```yaml
# 标题
site_name: 学习笔记

# 索引
nav:
  - 文档说明: index.md
  - 数学公式:
      - Latex: 数学公式/Markdown写作方法.md
  - 深度学习:
      - 深度学习: 深度学习/环境安装.md
  - 英语:
      - 名词: 英语/名词.md
      - 代词: 英语/代词.md

# 主题
theme:
  name: material
  features:
    - instant
    - navigation.tabs
    - navigation.sections

# 文本扩展
markdown_extensions:
  - pymdownx.arithmatex
  - pymdownx.superfences
  - pymdownx.highlight:
      linenums: true


# 额外的js
extra_javascript:
  - https://cdnjs.cloudflare.com/ajax/libs/mathjax/3.2.0/es5/tex-mml-chtml.js

# 额外的css
extra_css:
  - https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.3/css/all.min.css

# 额外的插件
extra:
  mathjax:
    tex:
      tags: 'ams'  # 启用 AMS 标签支持
      processEscapes: true

# 插件
plugins:
  - search
```
